# AI 思路总结

这是一台以《死亡笔记》为背景的 Linux 靶机。Nmap 扫描确定 80 和 22 端口后，我们绑定域名访问 Web 服务。通过目录遍历漏洞，我们在 WordPress 中成功提取到字典文件 `user.txt` 和 `notes.txt`。随后果断放弃低权限的 Web 后台爆破，直接使用 Hydra 对 SSH 进行凭据碰撞，成功拿到用户 `l` 的初始立足点。

在横向移动阶段，我们避开了破解数据库哈希获取 `www-data` 权限的“兔子洞”，转而在本地 `/opt` 目录深挖敏感信息。利用 CyberChef 将伪装文件 `case.wav` 经 Hex 转 Base64 连续解码，成功提取出密码 `kiraisevil`，水平越权至 `kira` 用户。

提权阶段极其直接，通过 `sudo -l` 发现 `kira` 拥有 `(ALL : ALL) ALL` 免密执行特权，一行 `sudo su` 即可拿下最高权限与 Root Flag。本靶机核心在于避开 Web 层的干扰项，依赖本地信息泄露与基础编码解码完成渗透，是锻炼战术决断与后渗透敏感度的极佳靶机。
