### Burp 代理转发配置

#### 配置 Upstream Proxy（代理中转）

1. 点击 Burp 顶部菜单：`User Options` → `Connections`
    
2. 找到 `Upstream Proxy Servers`，点击 `Add`
    
3. 填入如下内容：

    ```
    Destination host: 127.0.0.1 
    Destination port: 80 
    Proxy host: 192.168.240.144 
    Proxy port: 3218
    ```

4. 勾选 `Use this proxy server for all protocols`（可选）

👉 这表示：**凡是访问 `127.0.0.1:80` 的请求，Burp 都会通过 `192.168.240.144:3218` 转发。**

### 🧠 1. cron 的四大来源

在 Linux 中，cron 会从以下几个地方加载任务：

| 来源目录/命令                                           | 说明                                  |
| ------------------------------------------------- | ----------------------------------- |
| `/etc/crontab`                                    | 系统级任务，**格式需包含用户名**                  |
| `/etc/cron.d/`                                    | 额外的计划任务目录，**格式与 `/etc/crontab` 相同** |
| `/var/spool/cron/crontabs/`                       | 用户的 crontab（通过 `crontab -e` 添加）     |
| <br>`/etc/cron.daily/`, `/etc/cron.hourly/`, etc. | 定时脚本目录，由 `anacron` 等定期调用            |
